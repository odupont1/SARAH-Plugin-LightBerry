SARAH-Plugin-LightBerry
===========================

Le module "LightBerry" permet a Sarah de piloter le LightBerry via Hyperion-remote en SSH.

Exemple de quelques commandes :

    Sarah allume le Lightberry
	Sarah éteint le Lightberry
	Sarah reset le Lightberry
	Sarah Lightberry en rouge
	Sarah Lightberry en vert
	Sarah Lightberry en bleu
	Sarah Lightberry arc en ciel
	Sarah Lightberry k 2000
	Sarah Lightberry snake
